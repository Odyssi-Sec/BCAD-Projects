Description
This project is a web application that allows users to track and manage their academic
modules. 
It includes features such as user registration, login, module creation, editing, deletion, 
and a graphical representation of study hours.

Features
User Registration and Login:

New users can register with the system.
Registered users can log in securely.
Module Management:

Users can add new modules, specifying details such as code, name, credits, class hours per 
week, etc.
Existing modules can be edited, and users can record hours spent on each module.
Users can delete modules.
Graphical Representation:

The system provides a line graph displaying hours spent, self-study hours, 
and remaining hours for the current week.

Technologies Used:

ASP.NET Core
Entity Framework Core
Chart.js
HTML, CSS, JavaScript

Setup Instructions:

Clone the repository.
Install the necessary dependencies.
Configure the database settings.
Run the application.
Usage
Register a new account or log in with existing credentials.
Navigate to the module management section to add, edit, or delete modules.
Explore the graphical representation of study hours.
